## Про документ

## Правила оновлення

- При додаванні нової вимоги у [SSD](../architecture/SSD.md) — обов'язково додати тест-кейс у цю матрицю
- При зміні вимоги — переглянути відповідні тест-кейси та оновити опис
- При видаленні вимоги — позначити тест-кейси як `❌ Видалений`
- Статус `⚠️ Потребує оновлення` виставляється при зміні пов'язаної вимоги

---

*Матриця оновлюється синхронно із SSD. При розбіжності — SSD має пріоритет.*
