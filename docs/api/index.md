# Контрактна документація API AuthCore

**Роль API**: Internal (внутрішнє)  
**Тип**: Для frontend (SPA) та інших мікросервісів організації.

API надає контракт для:
- Реєстрації та автентифікації користувачів
- Управління JWT-токенами
- Двофакторної автентифікації
- Управління сесіями

Розроблено за підходом **API-first**. Всі зміни спочатку фіксуються в OpenAPI-специфікації.
