# Візуальна документація Frontend (Storybook)

## Компонент: LoginForm

### Stories:

**Default** — форма з полями email та password  
**Loading** — стан під час відправки запиту  
**Error** — відображення помилки (невірний пароль / 2FA required)  
**Success** — після успішного входу (перехід на головну сторінку)

## Компонент: 2FA Verification Modal

- **Default** — поле для введення 6-значного TOTP-коду
- **Loading** — перевірка коду
- **Error** — неправильний код
- **Success** — 2FA увімкнено

У реальному проєкті ці компоненти документувалися б у Storybook.js з інтерактивними прикладами.
